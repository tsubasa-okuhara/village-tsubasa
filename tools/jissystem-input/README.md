# 障害福祉サービス簡易入力ソフト 半自動入力ツール

居宅介護（家事援助）の月次入力作業を Excel + VBA + HiMacroEx で半自動化するための最小キット。

## 構成

```
tools/jissystem-input/
├── README.md                 ← このファイル
├── vba/
│   └── Module_AutoInput.bas  ← Excel に import する VBA モジュール
└── himacro/
    ├── 01_select_user.hmx    ← (奥原さんが Windows で録画して作成)
    ├── 02_input_date.hmx
    ├── 03_select_kaji.hmx
    ├── 04_input_start.hmx
    ├── 05_input_end.hmx
    └── 06_save.hmx
```

## セットアップ手順（Windows マシン）

### 1. ファイル一式を Windows に転送
このフォルダごと Windows マシンにコピー。例: `C:\jissystem-input\`

### 2. HiMacroEx をインストール
[HiMacroEx 公式](https://forest.watch.impress.co.jp/library/software/himacroex/) からダウンロード→インストール。

### 3. Excel ブックを準備（テンプレ）

新規 Excel ブックを作って **`.xlsm` 形式で保存** （マクロ有効）。
シート名: `Sheet1`（任意）

#### 列構成

| 列 | 内容 | データ例 |
|---|---|---|
| A | 利用者名 | 山田太郎 |
| B | サービス日 | 2026/05/01 |
| C | 開始時間 | 09:00 |
| D | 終了時間 | 10:00 |
| E | サービス種別 | 家事援助 |
| F | 備考 | 掃除・洗濯 |
| G | 受給者番号 | 0700001234 |
| H | 状態 | (空 / 未処理 / 処理中 / 完了 / エラー) |
| I | 実行日時 | (自動入力) |
| J | エラーメッセージ | (自動入力) |

1 行目はヘッダー、2 行目以降がデータ。

### 4. VBA モジュールを Excel に import

Excel で `Alt+F11` → VBA エディタが開く → 左ペインのプロジェクトを右クリック → 「ファイルのインポート」 → `vba/Module_AutoInput.bas` を選択。

### 5. 設定値を VBA で確認

`Module_AutoInput.bas` の冒頭にある定数を環境に合わせて編集:
- `HIMACRO_DIR`: HiMacroEx スクリプトの保存先（例: `C:\jissystem-input\himacro`）
- `HIMACRO_EXE`: HiMacroEx 実行ファイル（例: `C:\Program Files\HiMacroEx\HiMacroEx.exe`）

### 6. HiMacroEx スクリプトを録画

詳細は次セクション。

## HiMacroEx スクリプトの録画手順

事前準備として、**手動で 1 件だけ簡易入力ソフトに入力**して、各ステップで使えるショートカットキーを把握してください。マウスクリックは可能な限り避け、キー操作（Tab / Enter / Alt+X 等）で完結する経路を見つける。

### 各スクリプトの役割と推奨記録内容

| ファイル | 何をするか | 録画イメージ |
|---|---|---|
| `01_select_user.hmx` | 利用者検索を開いて、クリップボードの受給者番号で検索 + 選択 | `Alt+R`（仮）→ `Ctrl+V` → `Enter` → `Enter` |
| `02_input_date.hmx` | 日付フィールドへ移動して、クリップボードの YYYYMMDD を入力 | `Tab` × N → `Ctrl+V` → `Tab` |
| `03_select_kaji.hmx` | サービス種別ドロップダウンで「家事援助」を選択 | `Tab` → `か` 入力 → `Enter` （or 矢印キー） |
| `04_input_start.hmx` | 開始時間フィールドへ移動して、クリップボードの HHMM を入力 | `Tab` → `Ctrl+V` → `Tab` |
| `05_input_end.hmx` | 終了時間フィールドへ移動して、クリップボードの HHMM を入力 | `Tab` → `Ctrl+V` → `Tab` |
| `06_save.hmx` | 保存ボタン押下 + 確認ダイアログ Enter + 入力画面に戻る | `Alt+S`（仮）→ `Wait` → `Enter` |

⚠️ ショートカットキーは簡易入力ソフトの実際のメニューから事前確認してください（`Alt` を押すとメニューに下線アクセスキーが出ます）。

### 録画手順（HiMacroEx）

1. 簡易入力ソフトを起動して **実績入力画面** を表示
2. HiMacroEx を起動 → 「録画」ボタンクリック
3. 上の表の操作を順番に実行
4. 「停止」ボタン → ファイル名に `01_select_user.hmx` を指定して保存
5. 各ステップで同様に録画 → 6 ファイル作成
6. すべて `tools/jissystem-input/himacro/` に保存

### 動作確認（録画したスクリプトを単体テスト）

各 .hmx を **HiMacroEx で再生**して、想定通り動くか確認:
- `01_select_user.hmx`: クリップボードに受給者番号（例: `0700001234`）を手動でコピー → スクリプト再生 → 該当利用者が選択される？
- 各スクリプトで同様

## 実行手順（Phase 1: 1 件だけ）

### 準備
1. Excel に 1 行だけダミーデータを入れる
2. 簡易入力ソフトを起動して実績入力画面を表示（前面に出す）
3. Excel に戻る

### 実行
1. 対象行の任意のセルを選択（カーソルをその行に置く）
2. `Alt+F8` → マクロ一覧 → `ProcessOne` を実行
3. または VBA エディタから `ProcessOne` を実行

### 結果確認
- H 列「状態」が `完了` になっていれば成功
- `エラー` の場合、J 列に詳細
- 簡易入力ソフトで実際に入力されたか目視確認

## トラブルシュート

| 症状 | 対処 |
|---|---|
| 「クリップボードに値が入らない」 | VBA の `PutClipboard` を確認。MSForms 参照が必要なら、VBA エディタ → ツール → 参照設定 → 「Microsoft Forms 2.0 Object Library」を有効化 |
| 入力が速すぎて値が落ちる | 各 `RunHiMacro` の後の `Wait` 値を増やす（500 → 1000 ms） |
| ウィンドウが切り替わって他アプリに入力される | HiMacroEx 内で `WaitImage` などウィンドウ確認を入れる、または各ステップ前に `WIN_ACTIVATE` |
| 利用者が見つからない | 受給者番号のフォーマット確認。マスタ未登録の可能性も |
| 同名利用者が選ばれる | 受給者番号での検索に切替（`01_select_user.hmx` を改修） |

## 注意点

- **マウス座標は使わない**: 画面サイズや解像度で壊れる。必ずキー操作で記録
- **Tab × N の N**: 画面改修で順序が変わると壊れる。HiMacroEx スクリプトを録画し直す
- **データ形式は手動 1 件で確認してから自動化**: `:` 必要か、和暦/西暦どちらか、等
- **テスト環境推奨**: 本番データで試す前に、テスト利用者で動作確認
