# HiMacroEx 用 Excel DL 使い方ガイド

> **対象:** 居宅介護の月次入力を簡易入力ソフトに転記する人（奥原翼さん向け）
> **目的:** village-admin で生成した Excel を、Windows + HiMacroEx で簡易入力ソフトに半自動入力する全手順

---

## 全体の流れ（5 分で把握）

```
┌─────────────────────────────────────────────────────────┐
│ Mac (普段の作業 PC)                                       │
│  ① village-admin で「HiMacroEx 用 Excel DL」をクリック    │
│  ② Excel ファイル(.xlsx) がダウンロードされる             │
└──────────────┬──────────────────────────────────────────┘
               │ Excel をメール添付や OneDrive 等で Windows へ移動
               ▼
┌─────────────────────────────────────────────────────────┐
│ Windows (簡易入力ソフトがある PC)                         │
│  ③ Excel を開く（マクロ有効ブック .xlsm に変換）          │
│  ④ 簡易入力ソフトを起動 → 実績入力画面を表示             │
│  ⑤ Excel から `ProcessAll` マクロを実行                   │
│  ⑥ HiMacroEx が自動で簡易入力ソフトに転記                 │
└─────────────────────────────────────────────────────────┘
```

## 必要なもの一覧

### Mac 側
- ブラウザ（Chrome / Safari どちらでも）
- village-admin にログイン可能な Google アカウント

### Windows 側
| ソフト | 用途 | ダウンロード先 |
|---|---|---|
| **HiMacroEx** | キーボード/マウス操作の自動再生 | https://forest.watch.impress.co.jp/library/software/himacroex/ |
| **Microsoft Excel** | データ管理 + VBA マクロ実行 | Microsoft 365 など |
| **簡易入力ソフト** | 大田区への提出用ソフト（既にあるはず） | 国保連からの配布物 |

> 注: HiMacroEx は無料、Excel は有料（既存契約があるはず）。

## 詳細手順

### ステップ 1: village-admin で Excel を DL

1. Mac で https://village-admin-bd316.web.app/records-home.html を開く
2. ログイン（admin@village-support.jp）
3. 年・月を入力（例: 2026 / 5）→ 「利用者一覧を取得」
4. 「**HiMacroEx 用 Excel DL**」ボタンをクリック
5. ダウンロードフォルダに `居宅介護_HiMacroEx入力_令和8年5月.xlsx` ができる

### ステップ 2: Excel ファイルを Windows に移動

おすすめ：
- **OneDrive / Google Drive 経由** が一番楽
- もしくはメール添付で自分宛に送信

### ステップ 3: Windows で Excel を開く

1. Excel で当該ファイルを開く
2. **必ず `.xlsm` 形式で保存し直す**（マクロ有効ブック）
   - ファイル → 名前を付けて保存 → ファイル形式「Excel マクロ有効ブック (*.xlsm)」
3. VBA モジュールを import:
   - `Alt+F11` で VBA エディタを開く
   - 左ペインのプロジェクトを右クリック → 「ファイルのインポート」
   - `Module_AutoInput.bas` を選択
4. VBA 上部の定数 `HIMACRO_DIR` と `HIMACRO_EXE` のパスを Windows に合わせて編集
   - 例: `C:\jissystem-input\himacro\` と `C:\Program Files\HiMacroEx\HiMacroEx.exe`

### ステップ 4: 簡易入力ソフトを準備

1. 簡易入力ソフトを起動
2. ログイン → 該当事業所を選択
3. 「実績入力」画面を開いて、データ入力できる状態にする
4. ウィンドウを **画面手前** に表示（Excel と並べる or 切替で OK）

### ステップ 5: マクロ実行

#### 1 件だけテスト（最初は必ずこれ）

1. Excel に戻る
2. 1 行目（利用者の最初の行）の任意のセルをクリック
3. `Alt+F8` → マクロ一覧 → `ProcessOne` を選択 → 実行
4. 簡易入力ソフトに自動入力される様子を確認
5. H 列「状態」が **完了** になれば成功

#### 全件まとめて入力

1. `Alt+F8` → `ProcessAll` を選択 → 実行
2. 順次入力されていく（数十秒〜数分）
3. 全行の H 列が **完了** になれば終了

#### エラーが出たら

- H 列が「**エラー**」になる
- J 列にエラー詳細が書かれる
- そこの行から `ProcessAll` を再実行すると未処理分から再開

### ステップ 6: 簡易入力ソフトで保存

入力が終わったら、簡易入力ソフトの「保存」ボタンを押して確定。
（HiMacroEx は入力までで止める設計。確定操作は手動で確認するため）

## トラブルシュート

### A. HiMacroEx が起動しない / ファイルが見つからない

→ VBA の `HIMACRO_EXE` のパスが正しいか確認。ファイルエクスプローラで探して右クリック → プロパティでパス確認。

### B. 入力が速すぎて値が抜ける

→ VBA の各 `Wait` 値を `500` → `1000` に増やす。

### C. 違うウィンドウに入力される

→ 簡易入力ソフトを **必ず最前面** にしてからマクロ実行。または HiMacroEx スクリプトに `WIN_ACTIVATE` を追加。

### D. 利用者が見つからない

→ Excel の G 列「受給者番号」が正しいか確認。簡易入力ソフトに該当利用者が登録されているか。

### E. 既に入力済みのデータを誤って二重登録した

→ 簡易入力ソフトの「保存」を押す前なら取り消せる。押した後は手動で削除。

### F. .hmx ファイルがない

→ 初回セットアップ。`tools/jissystem-input/README.md` の「録画手順」セクションに従って 6 ファイル録画してください。

## 安全運用のコツ

| やる | やらない |
|---|---|
| ✅ 1 件 (`ProcessOne`) でテストしてから全件 | ❌ いきなり `ProcessAll` |
| ✅ 月初の少件数のうちに動作確認 | ❌ 月末にまとめて初めて使う |
| ✅ 簡易入力ソフトの保存前に必ず目視確認 | ❌ 入力即保存 |
| ✅ Excel ファイルは月ごとに別保存 | ❌ 1 ファイルを使い回し（H 列状態が混在）|

## 注意

- HiMacroEx は **画面解像度・ウィンドウ位置** に依存する場合あり。Windows 設定変更後は録画し直しが必要なことも。
- 簡易入力ソフトのバージョンが上がって UI が変わったら、`.hmx` を録画し直し。
- 自動入力中は **PC を触らない**（マウス・キーボード操作が干渉する）。
